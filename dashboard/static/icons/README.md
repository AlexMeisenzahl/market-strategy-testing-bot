# PWA Icons

Place the following icon files in this directory:

- **icon-192.png**: 192x192 pixels
- **icon-512.png**: 512x512 pixels
- **badge-72.png**: 72x72 pixels (for notification badge)

These should be PNG format with transparent background where appropriate.

## Generating Icons

You can generate icons from a source image using online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/
- Or create them manually with image editing software

## Design Guidelines

- Use high-contrast colors for visibility
- Keep the design simple and recognizable at small sizes
- Consider using the app's logo or a chart/trading symbol
- Ensure icons work well on both light and dark backgrounds
